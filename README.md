# Astro Shop

## Installation

```bash
# BUN
npm install

# YARN
yarn i

# PNPM
pnpm i

# BUN
bun i
```

## Development Mode

```bash
# BUN
npm run dev

# YARN
yarn dev

# PNPM
pnpm dev

# BUN
bun dev
```

## Production Build

```bash
# BUN
npm run build

# YARN
yarn run build

# PNPM
pnpm run build

# BUN
bun run build
```

## Preview

Run this command to check production build.

__NOTE: In order to run this command, you must build your application before.__

```bash
# BUN
npm run preview

# YARN
yarn preview

# PNPM
pnpm preview

# BUN
bun preview
```
